## Design Context

### Users
Potential employers, design directors, and creative collaborators viewing 蔡言's personal portfolio. Typically viewed on desktop in a focused browse session — someone deciding whether to reach out. Mobile use is secondary but must work well.

### Brand Personality
**Editorial · Precise · Bilingual**
The work is Swiss-influenced: strict grid, tight tracking, bold type contrasts. The bilingual Chinese/English framing is intentional — it's part of the identity, not an afterthought. The teal accent (#14b8a6) is the only color permission; everything else is black or white.

### Aesthetic Direction
- Swiss International Style: asymmetric grid, heavy sans-serif, clear hierarchy
- Black-and-white dominant with a single orange accent used sparingly (`oklch(68% 0.18 52)` / `--color-orange`)
- Type as texture: the oversized headline IS the hero image
- Light mode (white sections) transitioning to dark (black sections) — the scroll is a deliberate reveal
- Anti-references: anything "portfolio template", gradient text, glassmorphism, card-grid sameness

### Design Principles
1. **Type does the heavy lifting** — no decorative illustrations or icons
2. **Restraint in color** — orange (`--color-orange`) only where it earns its place
3. **Grid asymmetry over symmetry** — the indented second headline line is intentional; preserve it
4. **Animate purposefully** — entrances reveal editorial timing, not showboating
5. **Bilingual parity** — Chinese text should be typographically equal to English, not subordinate

### Mobile Adaptation Notes
- **ProjectSection**: Section uses `min-h-screen md:h-screen` so content can breathe on small devices without iOS Safari clipping; card image switched to `aspect-[4/3]` (was `16/9`) on mobile to reduce vertical footprint; `gap-8 md:gap-0` separates content from image on mobile
- **Footer**: Horizontal padding is `px-5 sm:px-8 md:px-10` — the full `px-10` caused the subtitle to overflow on 375px screens
- **SkillsSection**: Tool name (Figma, AE, etc.) shown as a compact label beneath "技能 / Skills" on mobile via `mobileToolPanelRefs`, animated by the same GSAP scroll-index logic as the desktop tool column
- **Breakpoints in use**: xs (400px custom), sm (640px), md (768px), lg (1024px)
